# 🚀 AI Career Predictor

AI-Powered Career Prediction & Resume Analysis Platform

CareerPilot AI is an intelligent career guidance system built using Machine Learning and Streamlit.  
It analyzes resumes and user-entered skills to predict suitable career paths, generate career roadmaps, analyze skill gaps, estimate salary ranges, and provide professional resume improvement suggestions.
